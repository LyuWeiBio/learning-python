# Python 从入门到机器学习实战（中文教程）

> 一套面向零基础学习者的系统化中文 Python 教程，从最基础的语法开始，一步步走到**机器学习实战**。
> 每一章约 **45 分钟** 学习时间，每一章都配有**练习题 + 参考答案**。

---

## 这套教程适合谁？

- **完全零基础**：没写过一行代码，想系统学习编程。
- **有一点基础**：会一点其它语言或 Python，想把知识补齐、打通到数据分析和机器学习。
- **转行 / 求职**：希望按照一条清晰的路径，最终能独立完成一个机器学习项目。

只要你能坚持每天学 1~2 章、动手敲代码并完成练习，学完这套教程你将能够：

- 熟练掌握 Python 语法与面向对象编程；
- 使用 NumPy / Pandas / Matplotlib 进行数据分析与可视化；
- 使用 scikit-learn 完成回归、分类、聚类等机器学习任务；
- 独立完成一个「从原始数据到训练好的模型」的完整项目。

---

## 如何使用本教程

1. **按顺序学习**：章节之间有依赖关系，建议从第 1 章开始逐章推进。
2. **动手敲代码**：不要只看，一定要亲手把每段代码敲一遍、跑一遍、改一改。
3. **先做题再看答案**：每章末尾的练习题请先独立完成，再展开「参考答案」核对。
4. **控制节奏**：每章约 45 分钟。累了就休息，理解比速度更重要。

> 关于环境安装（Python、编辑器、第三方库），请先阅读 **第 1 章**。

## 环境与依赖

- **Python 版本**：建议 3.10 及以上（本教程基于 Python 3.12 编写）。
- **第一、二部分**：只需标准的 Python 环境，无需第三方库。
- **第三、四部分**：需要安装数据科学相关库，见 [`requirements.txt`](requirements.txt)。

一键安装第三方库（建议在虚拟环境中执行，详见第 18 章）：

```bash
pip install -r requirements.txt
```

`代码示例/` 目录中提供了部分章节可直接运行的完整脚本。

---

## 学习建议

- **保持手感**：每天至少敲一段代码，哪怕只有 15 分钟。
- **善用报错**：报错信息是你最好的老师，学会阅读 Traceback（第 13 章会专门讲）。
- **构建作品集**：从第三部分开始，把每章的练习整理成小项目，逐渐形成自己的作品集。
- **官方文档**：遇到不懂的函数，优先查阅 [Python 官方文档](https://docs.python.org/zh-cn/3/) 与各库官方文档。
