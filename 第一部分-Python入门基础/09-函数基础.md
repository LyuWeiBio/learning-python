## 9.1 为什么需要函数

假设你要在程序里多次计算“圆的面积”，如果每次都写一遍 `3.14159 * r * r`，不仅啰嗦，改起来还容易出错。**函数**能把一段代码打包起来，起个名字，需要时“调用”它即可。

函数带来的好处：

- **复用**：写一次，到处调用。
- **清晰**：给一段逻辑起个有意义的名字，代码像读文章。
- **易维护**：逻辑集中在一处，改一处即全局生效。

其实我们早就用过函数了：`print()`、`len()`、`input()`、`int()` 都是 Python **内置函数**。本章学习如何定义**自己的**函数。

---

## 9.2 定义与调用函数

用关键字 `def` 定义函数：

```python
def greet():
    print("你好！")
    print("欢迎学习 Python")

# 定义之后要“调用”它才会执行
greet()        # 调用
greet()        # 可以反复调用
```

结构解读：

- `def` 是定义函数的关键字。
- `greet` 是函数名（命名规则同变量：小写 + 下划线）。
- 一对括号 `()` 用来放参数（这里暂时为空）。
- 冒号 `:` 之后，缩进的部分是**函数体**。
- 定义不会执行函数体；只有**调用** `greet()` 时才执行。

---

## 9.3 参数：给函数传入数据

参数让函数能处理不同的数据。括号里写的叫**形参**，调用时传入的叫**实参**：

```python
def greet(name):            # name 是形参
    print(f"你好，{name}！")

greet("小明")               # "小明" 是实参 → 你好，小明！
greet("小红")               # 你好，小红！
```

可以有多个参数，用逗号分隔：

```python
def add(a, b):
    print(a + b)

add(3, 5)         # 8
```

调用时，实参默认**按位置**依次对应形参（第一个给第一个，第二个给第二个）。

---

## 9.4 返回值：return

上面的函数只是打印，没有把结果“交回来”。用 `return` 可以让函数**返回一个结果**，供后续使用：

```python
def add(a, b):
    return a + b

result = add(3, 5)          # 把返回值存进变量
print(result)               # 8
print(add(10, 20) * 2)      # 返回值可以直接参与运算 → 60
```

### return 与 print 的区别（重点！）

初学者最容易混淆这两个：

- `print()` 只是把内容**显示**到屏幕上，程序拿不到这个值。
- `return` 是把结果**返回**给调用处，可以被存储、计算、传递。

```python
def add_print(a, b):
    print(a + b)            # 只显示，不返回

def add_return(a, b):
    return a + b            # 返回结果

x = add_print(3, 5)        # 屏幕显示 8，但 x 是 None！
y = add_return(3, 5)        # y 得到 8，可继续使用
```

> **关键理解**：没有 `return` 的函数，默认返回 `None`。如果你想“用函数算出的结果”，就必须 `return`。

### return 会立即结束函数

一旦执行到 `return`，函数立刻结束，后面的代码不再执行：

```python
def check(score):
    if score >= 60:
        return "及格"
    return "不及格"          # 只有前面没 return 才会走到这

print(check(80))            # 及格
```

## 9.5 返回多个值

`return` 后面用逗号可以返回多个值，本质是返回一个**元组**，配合解包很好用：

```python
def min_max(numbers):
    return min(numbers), max(numbers)

low, high = min_max([3, 7, 1, 9, 4])
print(low, high)            # 1 9
```

---

## 9.6 默认参数

给形参一个默认值，调用时如果不传，就用默认值：

```python
def greet(name, greeting="你好"):
    print(f"{greeting}，{name}！")

greet("小明")                    # 你好，小明！（用默认值）
greet("小红", "早上好")          # 早上好，小红！（覆盖默认值）
```

> **规则**：有默认值的参数必须放在**没有默认值的参数后面**。`def f(a=1, b)` 是错误的。

> **一个著名的坑**：不要用可变对象（如列表 `[]`）作默认值，比如 `def f(lst=[])`。这会导致多次调用共享同一个列表。初学阶段记住“默认值用不可变类型”即可，进阶后再深究。

---

## 9.7 关键字参数

调用时可以用 `参数名=值` 的形式明确指定，这叫**关键字参数**，好处是不用记参数顺序、可读性强：

```python
def create_user(name, age, city):
    print(f"{name}, {age}岁, 来自{city}")

# 按位置传
create_user("小明", 18, "北京")

# 用关键字传，顺序可以打乱
create_user(age=18, city="上海", name="小红")
```

位置参数必须在关键字参数**之前**。

---

## 9.8 文档字符串（docstring）

在函数体的第一行写一个字符串，用来说明函数的作用，这叫**文档字符串**。它是好习惯，也能被工具自动提取成文档：

```python
def circle_area(radius):
    """计算圆的面积。

    参数:
        radius: 圆的半径
    返回:
        面积（浮点数）
    """
    return 3.14159 * radius ** 2

print(circle_area.__doc__)      # 可以查看文档
help(circle_area)               # 或用 help() 查看
```

## 9.9 一个完整例子

把前面知识综合起来，写一个成绩处理函数：

```python
def analyze_scores(scores):
    """接收成绩列表，返回(平均分, 最高分, 及格率)。"""
    average = sum(scores) / len(scores)
    highest = max(scores)
    passed = len([s for s in scores if s >= 60])
    pass_rate = passed / len(scores)
    return average, highest, pass_rate

data = [88, 55, 92, 70, 45]
avg, high, rate = analyze_scores(data)
print(f"平均分：{avg:.1f}")
print(f"最高分：{high}")
print(f"及格率：{rate:.0%}")     # :.0% 会自动转成百分比
```

## 本章小结

- 用 `def 函数名(参数):` 定义函数，缩进部分是函数体；调用时才执行。
- 参数让函数处理不同数据；调用时按位置或按关键字传参。
- `return` 把结果交回调用处；**没有 return 的函数返回 `None`**。
- `return` 与 `print` 完全不同：一个返回值、一个只显示。
- `return a, b` 返回多个值（元组），可解包接收。
- 默认参数要放在后面；文档字符串（docstring）说明函数用途。
