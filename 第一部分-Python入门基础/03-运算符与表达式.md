# 第 03 章：运算符与表达式

## 本章学习目标

- 掌握算术运算符，尤其是整除 `//`、取余 `%`、幂 `**`。
- 掌握比较运算符，理解它们的结果是布尔值。

- 会用复合赋值运算符 `+=`、`-=` 等简化代码。

---

## 3.1 算术运算符

| 运算符 | 含义 | 例子 | 结果 |
| --- | --- | --- | --- |
| `+` | 加 | `3 + 2` | `5` |
| `-` | 减 | `3 - 2` | `1` |
| `*` | 乘 | `3 * 2` | `6` |
| `/` | 除（结果总是浮点数） | `7 / 2` | `3.5` |
| `//` | 整除（向下取整） | `7 // 2` | `3` |
| `%` | 取余（求模） | `7 % 2` | `1` |
| `**` | 幂（乘方） | `2 ** 10` | `1024` |

```python
print(7 / 2)     # 3.5    ← 注意：普通除法结果是浮点数
print(7 // 2)    # 3      ← 整除，只保留整数部分
print(7 % 2)     # 1      ← 余数
print(2 ** 10)   # 1024   ← 2 的 10 次方
```

### `//` 和 `%` 的实用场景

**取余 `%`** 常用于判断“能否整除”，比如判断奇偶：

```python
n = 17
print(n % 2)     # 1，余数不为 0，说明是奇数
```

**整除 `//`** 和 **取余 `%`** 经常搭配使用，比如把总秒数换算成分和秒：

```python
total_seconds = 125
minutes = total_seconds // 60    # 2 分
seconds = total_seconds % 60     # 5 秒
print(f"{minutes} 分 {seconds} 秒")   # 2 分 5 秒
```

> **注意负数的整除**：`//` 是**向下取整**（往更小的方向），所以 `-7 // 2` 结果是 `-4` 而不是 `-3`。初学阶段了解即可。

---

## 3.2 比较运算符

比较运算符用来比较两个值，结果永远是布尔值 `True` 或 `False`：

| 运算符 | 含义 |
| --- | --- |
| `==` | 等于（注意是两个等号！） |
| `!=` | 不等于 |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |

```python
print(3 == 3)    # True
print(3 != 5)    # True
print(3 > 5)     # False
print(3 <= 3)    # True
```

> **最常见的错误**：把“判断相等”的 `==` 写成了“赋值”的 `=`。请牢记：**一个等号是赋值，两个等号是比较**。

Python 还支持**连续比较**，非常符合数学直觉：

```python
age = 25
print(18 <= age < 60)    # True，等价于 (18<=age) and (age<60)
```

---

## 3.3 逻辑运算符

逻辑运算符用来组合多个条件，操作的是布尔值：

| 运算符 | 含义 | 说明 |
| --- | --- | --- |
| `and` | 与 | 两边**都**为真，结果才为真 |
| `or` | 或 | 只要**有一个**为真，结果就为真 |
| `not` | 非 | 取反，真变假、假变真 |

```python
age = 25
has_ticket = True

print(age >= 18 and has_ticket)    # True，两个条件都满足
print(age < 12 or has_ticket)      # True，第二个条件满足即可
print(not has_ticket)              # False，取反
```

真值表（记住这两句话就够了）：

- `and`：**一假则假**（有一个 False 结果就是 False）。
- `or`：**一真则真**（有一个 True 结果就是 True）。

### 短路求值（了解）

Python 的逻辑运算是“短路”的：`and` 遇到第一个假值就停止，`or` 遇到第一个真值就停止。这在后面写复杂条件时能避免不必要的计算，现在了解概念即可。

## 3.4 复合赋值运算符

在原值基础上做运算再赋回，可以用简写形式：

| 简写 | 等价于 |
| --- | --- |
| `x += 3` | `x = x + 3` |
| `x -= 3` | `x = x - 3` |
| `x *= 3` | `x = x * 3` |
| `x /= 3` | `x = x / 3` |
| `x //= 3` | `x = x // 3` |
| `x %= 3` | `x = x % 3` |
| `x **= 3` | `x = x ** 3` |

```python
score = 100
score -= 20      # 扣 20 分
print(score)     # 80
score += 5       # 加 5 分
print(score)     # 85
```

这种写法在循环里“累加计数”时非常常用（第 06 章会大量用到）。

---

## 3.5 运算符优先级

当多个运算符出现在一起时，Python 会按照**优先级**决定先算谁。大致顺序（从高到低）：

1. `**`（幂）
2. `*`、`/`、`//`、`%`（乘除类）
3. `+`、`-`（加减）
4. 比较运算符 `==`、`>`、`<` 等
5. `not`
6. `and`
7. `or`

```python
print(2 + 3 * 4)       # 14，先乘后加
print((2 + 3) * 4)     # 20，括号改变了顺序
print(2 ** 3 ** 2)     # 512，幂是从右往左算：2**(3**2)=2**9
```

> **黄金法则**：不确定优先级时，**直接加括号**。括号不仅能保证结果正确，还能让代码意图一目了然。没人会因为你多写括号而批评你。

---

## 3.6 表达式与语句

- **表达式**：能计算出一个值的代码，例如 `3 + 5`、`age >= 18`、`"a" + "b"`。
- **语句**：执行某个动作的完整指令，例如 `x = 10`（赋值语句）、`print(x)`（调用语句）。

一个赋值语句的右边通常就是一个表达式：

```python
total = price * count + shipping    # 右边是一个算术表达式
```

---

## 本章小结

- 算术：`+ - * /`，以及整除 `//`、取余 `%`、幂 `**`；普通除法 `/` 结果是浮点数。
- 比较：`== != > < >= <=`，结果是布尔值；支持 `18 <= x < 60` 的连续比较。
- 逻辑：`and`（一假则假）、`or`（一真则真）、`not`（取反）。
- 复合赋值：`+= -= *=` 等，简化“基于旧值更新”的写法。
- 优先级：`**` > 乘除 > 加减 > 比较 > 逻辑；**不确定就加括号**。

## 练习题

1. **年龄判断**：定义 `age = 20`，用一个表达式判断“是否成年且未满 60 岁”，打印布尔结果。
2. **时间换算**：给定 `seconds = 3725`，用 `//` 和 `%` 计算它等于多少小时、多少分、多少秒，输出 `1 小时 2 分 5 秒`。
3. **判断闰年**：给定年份 `year = 2024`，用逻辑运算符判断它是否为闰年并打印结果。规则：能被 4 整除但不能被 100 整除，**或者** 能被 400 整除。
4. **猜结果**：不运行，先写出下面每行的结果，再运行验证。

```python
print(10 % 3)
print(10 // 3)
print(2 ** 5)
print(5 > 3 and 2 > 4)
print(not (5 == 5))
```

5. **优先级**：给表达式 `2 + 3 * 4 > 10 or 1 == 2` 加上括号，明确表示 Python 实际的计算顺序，并说出最终结果。
