## 2.1 变量：给数据起个名字

**变量**就像一个贴了标签的盒子，用来存放数据。用等号 `=` 把右边的值“赋”给左边的名字：

```python
age = 18
name = "小红"
height = 1.68

print(name)     # 小红
print(age)      # 18
print(height)   # 1.68
```

这里的 `=` 不是数学里的“等于”，而是**赋值**：把右边的值存进左边的变量。

变量的值可以随时改变：

```python
score = 60
print(score)    # 60
score = 95      # 重新赋值，覆盖旧值
print(score)    # 95
```

也可以用旧值计算出新值：

```python
count = 10
count = count + 5   # 先算右边 10+5=15，再存回 count
print(count)        # 15
```

### 变量命名规则

**必须遵守（否则报错）：**

- 只能包含**字母、数字、下划线 `_`**，且**不能以数字开头**。
- 区分大小写：`age`、`Age`、`AGE` 是三个不同的变量。
- 不能使用 Python 的**关键字**（如 `if`、`for`、`class`）；内置函数名（如 `print`）也不要当作变量名重新赋值。

**建议遵守（好习惯）：**

- 使用**有意义**的英文名，如 `student_count` 而不是 `a`。
- 多个单词之间用下划线连接，全小写（这叫 `snake_case`，是 Python 的惯例）。

```python
# 推荐
student_count = 42
total_price = 99.9

# 不推荐（但合法）
a = 42
x1 = 99.9

# 错误示范（会报错）
# 2name = "小明"     # 不能以数字开头
# my-name = "小明"   # 不能用连字符
```

## 2.2 四种基本数据类型

Python 中最常用的四种基础类型：

| 类型 | 英文 | 说明 | 例子 |
| --- | --- | --- | --- |
| 整数 | `int` | 没有小数点的数 | `0`, `18`, `-100` |
| 浮点数 | `float` | 带小数点的数 | `3.14`, `-0.5`, `1.0` |
| 字符串 | `str` | 文本，用引号包裹 | `"你好"`, `'abc'` |
| 布尔值 | `bool` | 只有真/假两个值 | `True`, `False` |

```python
age = 18            # int
price = 9.9         # float
name = "Python"     # str
is_student = True   # bool
```

### 用 `type()` 查看类型

想知道一个值是什么类型，用内置函数 `type()`：

```python
print(type(18))         # <class 'int'>
print(type(3.14))       # <class 'float'>
print(type("你好"))     # <class 'str'>
print(type(True))       # <class 'bool'>
```

### 关于布尔值

布尔值只有两个：`True`（真）和 `False`（假），注意**首字母大写**。它常用来表示“是否成立”，是第 05 章条件判断的基础：

```python
is_adult = age >= 18
print(is_adult)     # True
```

## 2.3 类型转换

不同类型之间可以相互转换，常用的转换函数：

- `int(x)`：转成整数
- `float(x)`：转成浮点数
- `str(x)`：转成字符串

```python
# 字符串 → 数字
num_str = "100"
num = int(num_str)
print(num + 1)          # 101

# 数字 → 字符串
age = 18
age_str = str(age)
print("我今年" + age_str + "岁")   # 我今年18岁

# 整数 → 浮点数
print(float(5))         # 5.0

# 浮点数 → 整数（直接舍弃小数部分，不是四舍五入！）
print(int(3.99))        # 3
```

> **易错点**：字符串和数字不能直接用 `+` 拼接。`"我今年" + 18` 会报错 `TypeError`，必须先把 `18` 用 `str()` 转成字符串，或者用后面讲的 f-string。

```python
# 错误：TypeError
# print("我今年" + 18)
```

## 2.4 读取用户输入：input()

`input()` 函数让程序暂停，等待用户从键盘输入，回车后把输入的内容作为**字符串**返回：

```python
name = input("请输入你的名字：")
print("你好，" + name)
```

运行效果（用户输入了“小明”）：

```
请输入你的名字：小明
你好，小明
```

> **超级重要**：`input()` 返回的**永远是字符串**，即使你输入的是数字！如果要做数学运算，必须先转换类型。

```python
age = input("请输入你的年龄：")   # 假设输入 18
# age 现在是字符串 "18"，不是数字！
age = int(age)                    # 转成整数
print("明年你" + str(age + 1) + "岁")
```

一个常见的错误：

```python
a = input("第一个数：")   # 输入 3
b = input("第二个数：")   # 输入 5
print(a + b)              # 输出 "35"，因为是字符串拼接，不是相加！
```

正确做法：

```python
a = int(input("第一个数："))
b = int(input("第二个数："))
print(a + b)              # 输出 8
```

## 2.5 输出：print() 的更多用法

### 打印多个值

`print()` 可以接收多个值，用逗号隔开，输出时**自动用空格分隔**：

```python
name = "小红"
age = 18
print("姓名", name, "年龄", age)   # 姓名 小红 年龄 18
```

### 自定义分隔符与结尾

- `sep` 参数：控制多个值之间的分隔符（默认空格）。
- `end` 参数：控制这一行结尾的字符（默认换行 `\n`）。

```python
print("2024", "01", "01", sep="-")   # 2024-01-01
print("不换行", end="")
print("接着这一行")                   # 不换行接着这一行
```

## 2.6 f-string：最推荐的格式化方式

把变量“嵌入”到字符串里，最优雅的方式是 **f-string**（格式化字符串）：在字符串前加字母 `f`，然后用 `{}` 包住变量名。

```python
name = "小明"
age = 18
print(f"我叫{name}，今年{age}岁")   # 我叫小明，今年18岁
```

`{}` 里还可以直接写表达式：

```python
a = 3
b = 5
print(f"{a} + {b} = {a + b}")       # 3 + 5 = 8
```

控制小数位数（保留 2 位小数）：

```python
pi = 3.1415926
print(f"圆周率约为 {pi:.2f}")        # 圆周率约为 3.14
```

> f-string 是 Python 3.6+ 引入的特性，简洁直观，本教程后续会大量使用它。请一定掌握。

## 本章小结

- **变量**用 `=` 赋值，命名用小写字母 + 下划线，要有意义。
- 四种基本类型：`int`、`float`、`str`、`bool`；用 `type()` 查看类型。
- 类型转换：`int()`、`float()`、`str()`；`int(3.99)` 是直接截断得 3。
- `input()` 读取输入，返回值**永远是字符串**，做运算要先转类型。
- `print()` 可打印多个值，支持 `sep`、`end` 参数。
- **f-string**（`f"...{变量}..."`）是最推荐的格式化输出方式。

```python
x = input("x = ")
y = input("y = ")

```
