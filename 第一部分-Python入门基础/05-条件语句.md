# 第 05 章：条件语句

> **所属部分**：第一部分 · Python 入门基础
> **预计学习时间**：45 分钟
> **前置知识**：第 03 章（比较与逻辑运算符）

## 本章学习目标

- 理解程序的“分支”思想：根据条件走不同的路。
- 掌握 `if`、`if-else`、`if-elif-else` 的写法。
- 理解 Python 用**缩进**表示代码块的规则。
- 掌握嵌套条件与三元表达式。
- 理解值的“真假性”（Truthy / Falsy）。

---

## 5.1 为什么需要条件语句

到目前为止，我们的程序都是从上到下**顺序执行**的。但现实中经常需要“**如果……就……否则……**”的判断，比如：

- 如果分数 ≥ 60，显示“及格”，否则显示“不及格”。
- 如果余额充足，就扣款，否则提示“余额不足”。

这就需要**条件语句**（也叫分支结构）。

---

## 5.2 最简单的 if

```python
age = 20
if age >= 18:
    print("你是成年人")
    print("可以独立签署合同")
print("程序结束")
```

结构解读：

- `if` 后面跟一个**条件**（结果为 `True` 或 `False` 的表达式）。
- 条件后面必须有一个**冒号 `:`**。
- 下一行开始**缩进**的代码，是“条件成立时才执行”的代码块。
- 缩进结束（回到左边）就表示条件语句结束了，`print("程序结束")` 无论如何都会执行。

如果条件为 `False`，缩进块里的代码会被整体跳过。

### 关于缩进（非常重要！）

Python 用**缩进**（而不是大括号 `{}`）来表示代码的从属关系。这是 Python 最独特的语法之一。

- 缩进通常用 **4 个空格**（VS Code 里按一次 Tab 键即可，它会自动转成空格）。
- 同一个代码块的缩进必须**一致**，否则会报 `IndentationError`。

```python
# 错误示范：缩进不一致会报错
if True:
    print("第一行")
      print("缩进多了，报错")   # IndentationError
```

---

## 5.3 if-else：二选一

`else` 表示“否则”，当 `if` 条件不成立时执行：

```python
score = 55
if score >= 60:
    print("及格")
else:
    print("不及格")
```

`else` 后面同样要跟冒号，且不需要（也不能）写条件。

---

## 5.4 if-elif-else：多选一

当有多个互斥的情况时，用 `elif`（else if 的缩写）：

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

执行逻辑：从上到下依次检查条件，**一旦有一个成立就执行对应代码块，然后整个结构结束**，后面的 `elif`/`else` 都不再检查。

> **顺序很重要**！上面如果把 `score >= 60` 放在最前面，那么 85 分也会先匹配到“及格”。写多分支时要注意条件的先后顺序。

`elif` 可以有任意多个，`else` 可以省略：

```python
weather = "雨"
if weather == "晴":
    print("出门散步")
elif weather == "雨":
    print("带把伞")
# 没有 else 也可以
```

## 5.5 嵌套条件

`if` 里面还可以再写 `if`，形成嵌套。注意缩进层层递进：

```python
age = 20
has_ticket = True

if age >= 18:
    if has_ticket:
        print("欢迎入场")
    else:
        print("请先购票")
else:
    print("未成年人谢绝入内")
```

很多嵌套可以用逻辑运算符 `and`/`or` 改写得更简洁：

```python
if age >= 18 and has_ticket:
    print("欢迎入场")
```

> **建议**：能用 `and`/`or` 表达的，尽量避免深层嵌套，代码会更清晰。

---

## 5.6 值的“真假性”（Truthy / Falsy）

`if` 后面不一定非得是比较表达式，任何值都能判断真假。以下值被当作 `False`（称为 **falsy**）：

- `False`
- 数字 `0`、`0.0`
- 空字符串 `""`
- 空列表 `[]`、空字典 `{}`、空元组 `()`（后面章节会学）
- `None`（表示“空”的特殊值）

其余几乎都是 `True`（truthy）。

```python
name = input("请输入名字：").strip()
if name:               # 等价于 if name != ""
    print(f"你好，{name}")
else:
    print("你没有输入名字")
```

这种“直接判断变量是否为空”的写法非常 Pythonic，很常见。

---

## 5.7 三元表达式（条件表达式）

当“二选一”很简单时，可以用一行搞定：

```
值A if 条件 else 值B
```

```python
score = 75
result = "及格" if score >= 60 else "不及格"
print(result)          # 及格

# 等价于：
# if score >= 60:
#     result = "及格"
# else:
#     result = "不及格"
```

三元表达式适合简单场景，逻辑复杂时还是用完整的 `if-else` 更易读。

---

## 5.8 pass 占位符

如果某个分支暂时什么都不想写，但语法上又必须有内容，可以用 `pass` 占位：

```python
if score >= 60:
    pass    # TODO: 以后再实现
else:
    print("不及格")
```

---

## 本章小结

- `if 条件:`，条件成立则执行缩进的代码块；缩进用 4 个空格且必须一致。
- `if-else` 二选一；`if-elif-else` 多选一，从上到下匹配，命中即止。
- 多分支时**条件顺序**很关键。
- 任何值都有真假性：`0`、`""`、空容器、`None` 为假，其余为真。
- 三元表达式 `A if 条件 else B` 适合简单的二选一。
- `pass` 用作空代码块的占位符。

## 练习题

1. **判断正负**：让用户输入一个整数，判断它是正数、负数还是零，分别打印。
2. **BMI 计算器**：输入身高（米）和体重（千克），计算 BMI = 体重 / 身高²，并按标准输出：`< 18.5` 偏瘦、`18.5~24` 正常、`24~28` 偏胖、`≥ 28` 肥胖。
3. **闰年判断**：输入年份，判断是否闰年（规则见第 03 章），用 `if` 打印结果。
4. **成绩等级**：输入 0~100 的分数，用 `if-elif-else` 输出等级 A(≥90)/B(≥80)/C(≥70)/D(≥60)/E(<60)。
5. **改写为三元表达式**：把下面代码改写成一行三元表达式。

```python
n = 7
if n % 2 == 0:
    label = "偶数"
else:
    label = "奇数"
print(label)
```
