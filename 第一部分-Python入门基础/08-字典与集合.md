## 8.1 字典：键值对的集合

**字典（dict）**存储的是**键值对（key-value）**，用花括号 `{}`，每一项是 `键: 值` 的形式。可以把它想象成一本真正的“字典”：通过“词条（键）”查找“释义（值）”。

```python
student = {
    "name": "小明",
    "age": 18,
    "major": "计算机"
}
print(student)
```

- **键（key）**：必须是唯一且不可变的（通常用字符串或数字，不能用列表）。
- **值（value）**：可以是任意类型。

与列表用“数字索引”不同，字典用**键**来存取数据：

```python
print(student["name"])     # 小明，用键取值
print(student["age"])      # 18
```

> 如果访问一个**不存在的键**，会报错 `KeyError`。

---

## 8.2 增、改、删

```python
student = {"name": "小明", "age": 18}

# 新增或修改：键存在就修改，不存在就新增
student["age"] = 19            # 修改
student["city"] = "北京"       # 新增
print(student)                 # {'name': '小明', 'age': 19, 'city': '北京'}

# 删除
del student["city"]            # 删除某个键
age = student.pop("age")       # 删除并返回该键的值
print(age)                     # 19
```

---

## 8.3 安全取值：get()

直接用 `字典[键]` 访问不存在的键会报错。用 `get()` 则更安全——键不存在时返回 `None`（或你指定的默认值），不会报错：

```python
student = {"name": "小明", "age": 18}

print(student.get("name"))          # 小明
print(student.get("phone"))         # None，不报错
print(student.get("phone", "未填"))  # 未填，指定默认值
```

判断键是否存在，用 `in`：

```python
if "age" in student:
    print("有年龄信息")
```

---

## 8.4 遍历字典

```python
student = {"name": "小明", "age": 18, "major": "计算机"}

# 遍历键（默认）
for key in student:
    print(key)                 # name age major

# 遍历值
for value in student.values():
    print(value)               # 小明 18 计算机

# 同时遍历键和值（最常用）
for key, value in student.items():
    print(f"{key} = {value}")
```

- `keys()`：所有键
- `values()`：所有值
- `items()`：所有 (键, 值) 对

`items()` 配合解包 `for key, value in ...` 是遍历字典的标准姿势。

---

## 8.5 字典的典型应用：计数

字典非常适合“统计频率”。比如统计一句话里每个字符出现的次数：

```python
text = "hello"
counter = {}
for ch in text:
    counter[ch] = counter.get(ch, 0) + 1
print(counter)     # {'h': 1, 'e': 1, 'l': 2, 'o': 1}
```

这里 `counter.get(ch, 0)` 的妙处：第一次遇到某字符时返回默认值 0，加 1 后存入；之后每次再遇到就在原基础上加 1。

## 8.6 字典推导式

和列表推导式类似，字典也有推导式，形式是 `{键: 值 for ...}`：

```python
# 生成 {1:1, 2:4, 3:9, 4:16, 5:25}
squares = {x: x ** 2 for x in range(1, 6)}
print(squares)

# 从两个列表构造字典
names = ["a", "b", "c"]
scores = [90, 85, 95]
result = {name: score for name, score in zip(names, scores)}
print(result)      # {'a': 90, 'b': 85, 'c': 95}
```

`zip()` 能把多个序列“拉链式”地配对，非常实用。

---

## 8.7 嵌套字典

字典的值可以是列表、字典等，用于表达复杂结构（这也是 JSON 数据的形态，第 17 章会用到）：

```python
students = {
    "s01": {"name": "小明", "scores": [90, 85]},
    "s02": {"name": "小红", "scores": [95, 92]},
}
print(students["s01"]["name"])        # 小明
print(students["s02"]["scores"][0])   # 95
```

## 8.8 集合：无序不重复的元素

**集合（set）**也用花括号 `{}`，但里面只有值、没有键值对。它的两大特点：**无序**、**自动去重**。

```python
s = {1, 2, 3, 3, 2, 1}
print(s)              # {1, 2, 3}，重复的自动去掉了

# 注意：空集合必须用 set()，因为 {} 是空字典！
empty = set()
```

**去重**是集合最常见的用途：

```python
nums = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(nums))     # 转成集合去重，再转回列表
print(unique)                # [1, 2, 3, 4]（顺序可能不同）
```

集合的增删：

```python
s = {1, 2, 3}
s.add(4)          # 添加
s.discard(2)      # 删除（元素不存在也不报错）
print(s)          # {1, 3, 4}
```

---

## 8.9 集合运算

集合支持数学上的交、并、差运算，非常直观：

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a & b)      # {3, 4}          交集（都有的）
print(a | b)      # {1,2,3,4,5,6}   并集（合起来）
print(a - b)      # {1, 2}          差集（a 有 b 没有）
print(a ^ b)      # {1,2,5,6}       对称差（只在一个里的）
```

判断成员用 `in`（集合的查找速度极快，远快于列表）：

```python
print(3 in a)     # True
```

---

## 8.10 四种容器对比

| 容器 | 符号 | 有序 | 可变 | 可重复 | 特点 |
| --- | --- | --- | --- | --- | --- |
| 列表 list | `[]` | 是 | 是 | 是 | 通用有序序列 |
| 元组 tuple | `()` | 是 | 否 | 是 | 不可变序列 |
| 字典 dict | `{k:v}` | 是* | 是 | 键唯一 | 键值映射，按键存取 |
| 集合 set | `{}` | 否 | 是 | 否 | 去重、集合运算 |

> \* 从 Python 3.7 起，字典会保持插入顺序。

---

## 本章小结

- 字典 `{键: 值}` 通过**键**存取值；访问不存在的键会 `KeyError`，用 `get()` 更安全。
- 增改：`d[key] = value`；删除：`del` / `pop()`；判断：`key in d`。
- 遍历：`for k, v in d.items()`；还有 `keys()`、`values()`。
- 字典擅长**计数/映射**；支持字典推导式；`zip()` 可配对多个序列。
- 集合 `{}` **无序去重**，空集合用 `set()`；支持交 `&`、并 `|`、差 `-` 运算，查找极快。

## 练习题

1. **通讯录**：创建一个字典存 3 个人的“姓名: 电话”，然后新增一个人，修改其中一个人的号码，最后遍历打印所有人。
2. **词频统计**：给定字符串 `"the quick brown fox the lazy dog the"`，统计每个单词出现的次数（提示：`split()` + 字典计数）。
3. **安全查询**：给定字典 `prices = {"苹果": 5, "香蕉": 3}`，让用户输入商品名，用 `get()` 查询价格，查不到就提示“无此商品”。
4. **列表去重并排序**：给定 `data = [3, 1, 2, 3, 1, 4, 2, 5]`，去重后按升序输出。
5. **共同好友**：`a = {"张三", "李四", "王五"}`，`b = {"李四", "王五", "赵六"}`，求出两人的共同好友，以及所有不重复的好友总数。
