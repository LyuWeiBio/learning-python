## 7.1 列表：有序的数据集合

前面我们的变量一次只能存一个值。**列表（list）**可以在一个变量里存放**一串有序的值**，用方括号 `[]` 表示，元素之间用逗号分隔：

```python
fruits = ["苹果", "香蕉", "橙子"]
numbers = [10, 20, 30, 40]
mixed = [1, "你好", 3.14, True]    # 列表可以混装不同类型
empty = []                        # 空列表
```

列表里可以放任意类型，甚至可以嵌套列表：

```python
matrix = [[1, 2, 3], [4, 5, 6]]   # 二维列表
```

---

## 7.2 访问与修改元素

列表的索引和切片规则与字符串完全一样（从 0 开始，支持负索引和切片）：

```python
fruits = ["苹果", "香蕉", "橙子", "葡萄"]
print(fruits[0])     # 苹果
print(fruits[-1])    # 葡萄
print(fruits[1:3])   # ['香蕉', '橙子']
```

**与字符串不同的是，列表是“可变的”**——可以直接修改某个位置的元素：

```python
fruits[1] = "西瓜"
print(fruits)        # ['苹果', '西瓜', '橙子', '葡萄']
```

（字符串不能这样改，`s[0] = "x"` 会报错。）

---

## 7.3 列表的常用方法

### 增加元素

```python
lst = [1, 2, 3]
lst.append(4)         # 在末尾追加一个元素
print(lst)            # [1, 2, 3, 4]

lst.insert(0, 99)     # 在索引 0 处插入
print(lst)            # [99, 1, 2, 3, 4]

lst.extend([5, 6])    # 把另一个列表的元素逐个追加
print(lst)            # [99, 1, 2, 3, 4, 5, 6]
```

> 注意区别：`append([5,6])` 会把 `[5,6]` 作为**一个元素**加进去；`extend([5,6])` 是把 5 和 6 **分别**加进去。

### 删除元素

```python
lst = [10, 20, 30, 20]
lst.remove(20)        # 删除第一个值为 20 的元素
print(lst)            # [10, 30, 20]

x = lst.pop()         # 弹出并返回最后一个元素
print(x)              # 20
print(lst)            # [10, 30]

y = lst.pop(0)        # 弹出指定索引的元素
print(y)              # 10

del lst[0]            # 用 del 按索引删除
```

### 查找与统计

```python
lst = [10, 20, 30, 20]
print(len(lst))       # 4，元素个数
print(20 in lst)      # True，判断是否存在
print(lst.index(30))  # 2，返回第一次出现的索引
print(lst.count(20))  # 2，统计出现次数
```

### 排序与反转

```python
nums = [3, 1, 4, 1, 5, 9, 2]
nums.sort()               # 原地升序排序（改变原列表）
print(nums)               # [1, 1, 2, 3, 4, 5, 9]

nums.sort(reverse=True)   # 降序
print(nums)               # [9, 5, 4, 3, 2, 1, 1]

nums.reverse()            # 反转顺序
```

> `sort()` 会**改变原列表**并返回 `None`。如果想保留原列表、得到一个新的排序结果，用内置函数 `sorted(nums)`。

```python
original = [3, 1, 2]
new = sorted(original)    # 不改变 original，返回新列表
print(original)           # [3, 1, 2]
print(new)                # [1, 2, 3]
```

## 7.4 遍历列表

```python
fruits = ["苹果", "香蕉", "橙子"]

# 直接遍历元素（最常用）
for fruit in fruits:
    print(fruit)

# 同时需要索引和元素时，用 enumerate
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")
# 0: 苹果
# 1: 香蕉
# 2: 橙子
```

`enumerate()` 会同时给出**索引**和**元素**，非常实用。

---

## 7.5 列表推导式（List Comprehension）

这是 Python 极具特色的语法，用一行代码从旧列表生成新列表。

**基本形式**：`[表达式 for 变量 in 序列]`

```python
# 生成 1~5 的平方
squares = [x ** 2 for x in range(1, 6)]
print(squares)        # [1, 4, 9, 16, 25]
```

它等价于：

```python
squares = []
for x in range(1, 6):
    squares.append(x ** 2)
```

**带条件筛选**：`[表达式 for 变量 in 序列 if 条件]`

```python
# 取出 1~20 中的偶数
evens = [x for x in range(1, 21) if x % 2 == 0]
print(evens)          # [2, 4, 6, ..., 20]

# 把字符串列表全部转成大写
words = ["hello", "world"]
upper = [w.upper() for w in words]
print(upper)          # ['HELLO', 'WORLD']
```

列表推导式简洁高效，是 Python 程序员的必备技能。初学时如果觉得绕，可以先用普通 `for` 循环写，理解后再逐步改用推导式。

## 7.6 元组：不可变的列表

**元组（tuple）**和列表很像，但用**圆括号 `()`**，而且**创建后不能修改**（不可变）：

```python
point = (3, 5)
print(point[0])      # 3，索引方式和列表一样
# point[0] = 10      # ✗ 报错！元组不能修改

colors = ("红", "绿", "蓝")
print(len(colors))   # 3
```

> 定义**单个元素**的元组时，必须加逗号：`(5,)`。`(5)` 只是数字 5 加了括号，不是元组。

**为什么需要不可变的元组？**

- 表示“一组固定不变”的数据，如坐标 `(x, y)`、RGB 颜色 `(255, 0, 0)`。
- 更安全（防止被意外修改），性能也略优于列表。
- 可以作为字典的键（第 08 章会用到），而列表不行。

---

## 7.7 元组解包

可以把元组（或列表）里的多个值“一次性”赋给多个变量，这叫**解包**：

```python
point = (3, 5)
x, y = point          # 解包
print(x, y)           # 3 5
```

这其实就是第 02 章交换变量 `a, b = b, a` 背后的原理。函数返回多个值时也常用它（`def`/`return` 语法见第 09 章）：

```python
def min_max(nums):
    return min(nums), max(nums)   # 返回一个元组

low, high = min_max([3, 1, 4, 1, 5])
print(low, high)      # 1 5
```

---

## 7.8 列表 vs 元组：如何选择

| 特性 | 列表 list | 元组 tuple |
| --- | --- | --- |
| 符号 | `[]` | `()` |
| 可变性 | 可变（能增删改） | 不可变 |
| 用途 | 需要频繁修改的数据 | 固定不变的一组数据 |
| 能否作字典键 | 否 | 能 |

**经验法则**：默认用列表；只有当你确定这组数据不该被改变时，才用元组。

---

## 本章小结

- 列表 `[]` 是**有序可变**的集合，支持索引、切片，可混装类型、可嵌套。
- 增删：`append`/`insert`/`extend`，`remove`/`pop`/`del`；查找：`in`/`index`/`count`/`len`。
- 排序：`sort()` 原地排序（返回 None），`sorted()` 返回新列表。
- 遍历用 `for`，需要索引时用 `enumerate`。
- **列表推导式** `[表达式 for x in 序列 if 条件]` 一行生成新列表。
- 元组 `()` 是**不可变**的，适合固定数据；解包 `x, y = point` 很实用。
