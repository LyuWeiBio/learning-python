# 第 01 章：开发环境搭建与第一个程序

> **所属部分**：第一部分 · Python 入门基础
> **预计学习时间**：45 分钟
> **前置知识**：无，零基础即可开始

## 本章学习目标

- 了解 Python 是什么、能做什么、为什么值得学。
- 在自己的电脑上安装 Python，并验证安装成功。

## 1.1 Python 是什么

Python 是一门**通用编程语言**，由 Guido van Rossum 于 1991 年发布。它的设计哲学强调**代码的可读性**，语法简洁、接近自然语言，因此非常适合初学者。

Python 的应用非常广泛：

| 领域 | 举例 |
| --- | --- |
| 数据分析 / 数据科学 | 处理表格、统计分析、可视化 |
| 人工智能 / 机器学习 | 图像识别、推荐系统、大语言模型 |
| Web 后端开发 | 网站、API 服务（Django、Flask、FastAPI） |
| 自动化 / 脚本 | 批量处理文件、爬虫、办公自动化 |
| 科学计算 | 物理、生物、金融建模 |

> 本教程的最终目标是**机器学习实战**，而 Python 正是当今机器学习领域最主流的语言，几乎所有主流框架（scikit-learn、PyTorch、TensorFlow）都以 Python 为主要接口。

## 1.2 安装 Python

### 方法一：官方安装包（推荐初学者）

1. 打开浏览器访问官网：<https://www.python.org/downloads/>
2. 点击黄色的 **Download Python 3.x.x** 按钮（选择 3.10 及以上版本）。
3. 运行下载好的安装程序。

> **Windows 用户特别注意**：安装第一步的窗口底部，务必勾选 **“Add Python to PATH”（把 Python 添加到环境变量）**，然后再点击 “Install Now”。这一步能让你在命令行里直接使用 `python` 命令。

- **Windows**：勾选 “Add Python to PATH” → Install Now。
- **macOS**：下载 `.pkg` 安装包，一路“继续”即可；macOS 也可用 `brew install python`。
- **Linux**：大多数发行版自带 Python 3，也可用包管理器安装，例如 Ubuntu：`sudo apt install python3 python3-pip`。

### 验证安装是否成功

打开你的**终端**（命令行）：

- Windows：按 `Win + R`，输入 `cmd` 回车；或搜索“PowerShell”。
- macOS：打开“终端”(Terminal) 应用。
- Linux：`Ctrl + Alt + T`。

在终端中输入下面的命令并回车：

```bash
python --version
```

如果显示类似 `Python 3.12.3` 的版本号，说明安装成功！

> **常见坑**：如果提示“找不到命令”，在 Windows 上试试 `py --version`，在 macOS/Linux 上试试 `python3 --version`。很多系统里 `python3` 才是正确的命令名。本教程后文统一写 `python`，你需要根据自己的系统替换成 `python3` 或 `py`。

## 1.3 两种运行 Python 的方式

### 方式一：交互式解释器（Interactive Mode）

在终端里直接输入 `python`（或 `python3`）回车，你会看到 `>>>` 提示符，这就是 Python 的**交互式解释器**，可以一行一行地即时执行代码：

```python
>>> 1 + 1
2
>>> print("你好")
你好
>>> 10 * 8
80
```

交互式解释器非常适合**做实验、验证想法**。输入 `exit()` 或按 `Ctrl + D`（Windows 上是 `Ctrl + Z` 再回车）退出。

### 方式二：运行脚本文件（Script Mode）

真正开发时，我们把代码写在一个以 `.py` 结尾的文件里，然后一次性运行整个文件。这就是**脚本模式**，也是本教程的主要方式。

假设有一个文件 `hello.py`，在终端里运行它的命令是：

```bash
python hello.py
```

## 1.4 安装代码编辑器（VS Code）

虽然记事本也能写代码，但一个专业的编辑器能极大提升效率（语法高亮、自动补全、一键运行）。推荐 **Visual Studio Code（VS Code）**，免费、跨平台、生态强大。

1. 访问 <https://code.visualstudio.com/> 下载并安装。
2. 打开 VS Code，点击左侧的**扩展**图标（四个方块），搜索 **“Python”**（Microsoft 出品），点击安装。
3. 通过菜单 `文件 → 打开文件夹`，选择一个你专门用来学习的文件夹（例如 `python-learning`）。

> **建议**：为本教程新建一个文件夹，把每章练习的代码都放进去，养成整理代码的好习惯。

## 1.5 你的第一个程序：Hello, World!

按照编程界的传统，第一个程序永远是打印一句 “Hello, World!”。

在你的学习文件夹里新建一个文件 `hello.py`，输入以下内容：

```python
print("Hello, World!")
print("我的第一个 Python 程序")
```

保存后，在终端中运行（先 `cd` 到文件所在目录）：

```bash
python hello.py
```

你会看到输出：

```
Hello, World!
我的第一个 Python 程序
```

🎉 恭喜！你已经成功运行了人生第一个 Python 程序。

### 拆解这行代码

```python
print("Hello, World!")
```

- `print` 是一个**函数**，作用是把内容“打印”到屏幕上。
- 一对小括号 `()` 表示“调用”这个函数。
- 括号里的 `"Hello, World!"` 是一段**字符串**（文本），必须用引号（单引号 `'` 或双引号 `"` 都行）包起来。

> **务必使用英文标点**！代码里的括号、引号、逗号都必须是英文半角符号。中文的 `（）"" ，` 会导致报错，这是初学者最常见的错误之一。

## 1.6 注释：写给人看的说明

**注释**是给人阅读的说明文字，Python 在运行时会完全忽略它。用 `#` 号开头表示单行注释：

```python
# 这是一行注释，程序不会执行它
print("Hello")  # 井号后面的内容也是注释

# 下面这行被注释掉了，不会运行
# print("我不会被打印")
```

多行说明可以用三引号 `"""` 包裹（严格来说它是一个“文档字符串”，但常被当多行注释用）：

```python
"""
这是一段
多行说明文字。
通常用于解释整个文件或函数的作用。
"""
print("正文代码")
```

**为什么要写注释？** 因为几个月后的你，可能会完全看不懂现在写的代码。好的注释解释“为什么这么做”，而不是简单复述“做了什么”。

---

## 1.7 关于报错：别怕，它是你的朋友

初学时你一定会遇到各种报错（Error），这非常正常。例如把 `print` 拼错成 `prin`：

```python
prin("Hello")
```

运行会得到：

```
NameError: name 'prin' is not defined
```

报错信息的最后一行通常告诉你**错误类型和原因**。学会阅读它，就能快速定位问题。我们在第 13 章会系统学习异常与报错处理，现在你只需记住：**报错不可怕，它在帮你找 bug**。

- Python 是一门简洁易读、应用广泛的语言，是数据分析与机器学习的首选。
- 从官网安装 Python，并用 `python --version` 验证。
