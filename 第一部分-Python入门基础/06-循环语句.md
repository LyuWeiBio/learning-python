## 6.1 为什么需要循环

如果要打印 1 到 100，难道要写 100 行 `print`？循环就是为了**重复执行**某段代码而生的。Python 有两种循环：`for` 和 `while`。

---

## 6.2 for 循环与 range()

`for` 循环用来**遍历**一个序列（例如一串数字、一个字符串、一个列表）中的每个元素：

```python
for ch in "abc":
    print(ch)
# 依次打印 a、b、c
```

要循环固定次数，最常用的是配合 `range()`：

### range() 的三种形式

```python
# 形式一：range(n) → 0, 1, 2, ..., n-1
for i in range(5):
    print(i)          # 0 1 2 3 4

# 形式二：range(start, stop) → start 到 stop-1
for i in range(1, 6):
    print(i)          # 1 2 3 4 5

# 形式三：range(start, stop, step) → 带步长
for i in range(0, 10, 2):
    print(i)          # 0 2 4 6 8
```

- `range` 同样**含头不含尾**（不包含 stop）。
- 步长可以是负数，用来倒数：`range(5, 0, -1)` → `5 4 3 2 1`。

打印 1 到 100 就变得很简单：

```python
for i in range(1, 101):
    print(i)
```

### 循环变量

`for i in ...` 里的 `i` 是**循环变量**，每次循环自动取序列里的下一个值。名字可以随便起，习惯上计数用 `i`、`j`；遍历有意义的数据时应起有意义的名字：

```python
for fruit in ["苹果", "香蕉", "橙子"]:
    print(f"我喜欢吃{fruit}")
```

---

## 6.3 经典模式：累加与计数

循环最常见的用途是“把一堆数据汇总成一个结果”。

**累加求和**（先定义一个“累加器”初始为 0，循环里不断加）：

```python
total = 0
for i in range(1, 101):
    total += i          # 等价于 total = total + i
print(total)            # 5050
```

**计数**（统计满足条件的个数）：

```python
count = 0
for i in range(1, 101):
    if i % 3 == 0:      # 能被 3 整除
        count += 1
print(f"1~100 中有 {count} 个数能被 3 整除")   # 33
```

> 记住这个套路：**循环外初始化变量 → 循环内更新变量 → 循环后使用结果**。它是解决无数问题的通用模板。

---

## 6.4 while 循环

`while` 循环在**条件为真时反复执行**，适合“不知道要循环多少次，但知道结束条件”的场景：

```python
count = 1
while count <= 5:
    print(count)
    count += 1          # 千万别忘了更新条件变量！
print("循环结束")
```

执行逻辑：先判断条件 → 为真则执行循环体 → 再回去判断条件……直到条件为假。

### ⚠️ 警惕死循环

如果循环条件永远为真，程序就会**永远运行下去**（死循环）。最常见的原因是忘了更新条件变量：

```python
# 危险！这是死循环
count = 1
while count <= 5:
    print(count)
    # 忘了写 count += 1，count 永远是 1，永远 <= 5
```

如果不小心运行了死循环，按 `Ctrl + C` 可以强制中断程序。

### while 的典型用法：等待合法输入

```python
while True:                       # 故意写成死循环
    age = input("请输入年龄(正整数)：")
    if age.isdigit():             # 判断字符串是否全是数字
        age = int(age)
        break                     # 输入合法，跳出循环
    print("输入无效，请重试！")
print(f"你的年龄是 {age}")
```

这里用了 `while True` + `break` 的组合，非常常见。

---

## 6.5 break 与 continue

- **`break`**：立即**跳出**整个循环。
- **`continue`**：跳过本次循环剩余的代码，直接进入**下一次**循环。

```python
# break：找到第一个能被 7 整除的数就停止
for i in range(1, 100):
    if i % 7 == 0:
        print(f"找到了：{i}")
        break

# continue：只打印奇数（遇到偶数就跳过）
for i in range(1, 11):
    if i % 2 == 0:
        continue
    print(i)          # 1 3 5 7 9
```

---

## 6.6 循环中的 else（了解）

Python 的循环可以带一个 `else` 子句，它在**循环正常结束（没有被 break 打断）**时执行：

```python
# 判断一个数是否为质数（本例要求 n ≥ 2；n = 1 时循环不执行，会误判为质数）
n = 13
for i in range(2, n):
    if n % i == 0:
        print(f"{n} 不是质数")
        break
else:
    print(f"{n} 是质数")    # 循环没被 break，说明没找到因子
```

这个特性不常用，看懂即可。

---

## 6.7 嵌套循环

循环里面可以再套循环。经典例子是打印乘法表：

```python
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}×{i}={i*j}", end="\t")
    print()          # 每行末尾换行
```

输出（九九乘法表）：

```
1×1=1
1×2=2   2×2=4
1×3=3   2×3=6   3×3=9
...
```

外层循环走一次，内层循环要完整走一遍。嵌套循环功能强大，但层数太多会变慢，一般不超过 2~3 层。

---

## 本章小结

- `for` 遍历序列；`range(n)` / `range(start, stop)` / `range(start, stop, step)` 生成数字序列，含头不含尾。
- `while 条件:` 在条件为真时反复执行，务必更新条件变量，避免死循环（`Ctrl+C` 中断）。
- 经典模式：循环外初始化累加器/计数器，循环内更新，循环后使用。
- `break` 跳出循环，`continue` 跳过本次。
- `while True` + `break` 常用于反复读取直到输入合法。
- 嵌套循环：外层一次、内层一轮。

1. **求和**：用 `while` 循环计算 1 到 100 的和（结果应为 5050）。
2. **阶乘**：用 `for` 循环计算 `n!`（n 的阶乘），例如输入 5 输出 120。

```
*
**
***
****
*****
```

4. **猜数字游戏**：程序预设答案 `answer = 7`，用 `while` 让用户反复猜，猜大提示“大了”，猜小提示“小了”，猜中提示“恭喜”并结束。
5. **统计数字**：对 1~50 的所有整数（用 `range` 生成），统计其中**既是偶数又能被 3 整除**的数有几个，并把它们打印出来。
