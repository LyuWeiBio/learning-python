## 4.1 创建字符串

字符串是用引号包裹的文本。单引号、双引号效果一样：

```python
s1 = '单引号字符串'
s2 = "双引号字符串"
```

如果字符串内部本身包含引号，可以用另一种引号包裹，避免冲突：

```python
s3 = "他说：'你好'"
s4 = '书名是《Python 入门》'
```

**多行字符串**用三引号 `"""` 或 `'''`：

```python
poem = """床前明月光，
疑是地上霜。
举头望明月，
低头思故乡。"""
print(poem)
```

---

## 4.2 转义字符

有些特殊字符需要用反斜杠 `\` 来表示，称为**转义字符**：

| 转义 | 含义 |
| --- | --- |
| `\n` | 换行 |
| `\t` | 制表符（Tab） |
| `\\` | 一个反斜杠 |
| `\"` | 一个双引号 |
| `\'` | 一个单引号 |

```python
print("第一行\n第二行")     # \n 会换行
print("姓名\t年龄")          # \t 是制表符，用于对齐
print("路径：C:\\Users")     # 想打印一个反斜杠要写两个
```

如果不希望反斜杠被当作转义（比如文件路径），可以用**原始字符串**，在引号前加 `r`：

```python
print(r"C:\Users\name\test")   # 原样输出，不转义
```

## 4.3 字符串拼接与重复

```python
first = "Py"
second = "thon"
print(first + second)      # Python，用 + 拼接
print("=" * 20)            # 打印 20 个等号，用 * 重复
print("哈" * 3)            # 哈哈哈
```

拼接变量时，更推荐用第 02 章学过的 **f-string**：

```python
name = "小明"
age = 18
print(f"{name} 今年 {age} 岁")
```

求字符串长度用 `len()`：

```python
print(len("Hello"))       # 5
print(len("你好世界"))     # 4，一个汉字算一个字符
```

---

## 4.4 索引：取出单个字符

字符串中的每个字符都有一个**位置编号（索引）**，从 `0` 开始：

```
 字符:  P   y   t   h   o   n
 正索引: 0   1   2   3   4   5
 负索引:-6  -5  -4  -3  -2  -1
```

```python
s = "Python"
print(s[0])      # P，第一个字符
print(s[3])      # h
print(s[-1])     # n，最后一个字符（负索引从右数）
print(s[-2])     # o
```

> 索引越界会报错：`s[100]` 会抛出 `IndexError`。

---

## 4.5 切片：取出一段子串

切片语法：`s[起始:结束]`，取出从“起始”到“结束**前一个**”的部分（**含头不含尾**）：

```python
s = "Python"
print(s[0:3])     # Pyt   （索引 0、1、2）
print(s[2:5])     # tho   （索引 2、3、4）
print(s[1:])      # ython （从 1 到末尾）
print(s[:3])      # Pyt   （从开头到 3 之前）
print(s[:])       # Python（整个复制一份）
print(s[-2:])     # on    （最后两个）
```

切片还可以带**步长**：`s[起始:结束:步长]`：

```python
s = "0123456789"
print(s[::2])     # 02468   每隔一个取一个
print(s[::-1])    # 9876543210  步长 -1 表示反转！
```

> `s[::-1]` 是反转字符串的经典写法，一定要记住。

## 4.6 常用字符串方法

字符串自带许多方法，用 `字符串.方法()` 的形式调用。**注意：字符串是不可变的，这些方法都返回新字符串，不会改变原字符串。**

### 大小写转换

```python
s = "Hello World"
print(s.upper())      # HELLO WORLD
print(s.lower())      # hello world
print(s.title())      # Hello World（每个单词首字母大写）
```

### 去除空白：strip

```python
s = "   hello   "
print(s.strip())       # "hello"，去掉两端空白
print(s.lstrip())      # 只去左边
print(s.rstrip())      # 只去右边
```

处理用户输入时，`strip()` 极其常用，可以去掉不小心多打的空格。

### 替换：replace

```python
s = "我爱Java，Java很好"
print(s.replace("Java", "Python"))   # 我爱Python，Python很好
```

### 查找与判断

```python
s = "hello@example.com"
print(s.find("@"))              # 5，返回第一次出现的索引；找不到返回 -1
print("@" in s)                 # True，用 in 判断是否包含
print(s.startswith("hello"))    # True
print(s.endswith(".com"))       # True
print("l".count("l"))           # 统计出现次数
print(s.count("l"))             # 2
```

### 分割：split

`split()` 按指定分隔符把字符串切成一个**列表**（列表见第 07 章）：

```python
line = "苹果,香蕉,橙子"
fruits = line.split(",")
print(fruits)        # ['苹果', '香蕉', '橙子']

sentence = "hello world python"
words = sentence.split()      # 不给参数默认按空白分割
print(words)         # ['hello', 'world', 'python']
```

### 拼接：join

`join()` 是 `split` 的“逆操作”，把列表用指定字符连接成字符串：

```python
fruits = ["苹果", "香蕉", "橙子"]
print("-".join(fruits))       # 苹果-香蕉-橙子
print("".join(fruits))        # 苹果香蕉橙子
```

## 4.7 综合小例子

判断一个邮箱字符串是否“看起来合法”（包含 `@` 和 `.`）：

```python
email = input("请输入邮箱：").strip()

if "@" in email and "." in email:
    user = email.split("@")[0]
    print(f"你好，{user}！邮箱格式看起来没问题。")
else:
    print("这似乎不是一个有效的邮箱。")
```

（`if/else` 是第 05 章内容，这里先感受一下字符串方法的威力。）

## 本章小结

- 字符串用引号创建，多行用三引号；转义字符 `\n`（换行）、`\t`（制表符）。
- `+` 拼接、`*` 重复、`len()` 求长度、`in` 判断包含。
- **索引**从 0 开始，负索引从 -1 开始；越界会 `IndexError`。
- **切片** `s[start:end:step]` 含头不含尾；`s[::-1]` 反转字符串。
- 常用方法：`upper/lower/strip/replace/find/split/join/startswith/count`。
- 字符串**不可变**，所有方法都返回新字符串。

## 练习题

1. **取子串**：给定 `s = "programming"`，分别用切片取出：前 4 个字符、后 3 个字符、整个反转。
2. **规范化姓名**：用户输入可能有多余空格和大小写混乱，例如 `"  jOHN  "`，请处理成首字母大写、去掉空格的 `"John"`。
3. **域名提取**：给定邮箱 `"student@pku.edu.cn"`，提取出 `@` 后面的域名部分 `"pku.edu.cn"`。
4. **单词计数**：给定句子 `"the quick brown fox jumps over the lazy dog"`，统计一共有多少个单词。
5. **回文判断**：判断字符串 `"level"` 是否是回文（正读反读都一样），打印 `True` 或 `False`。
