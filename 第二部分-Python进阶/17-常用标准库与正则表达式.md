## 17.1 json：数据交换的通用格式

JSON 是网络上最常见的数据格式（几乎所有 API 返回的都是 JSON）。它的结构和 Python 的字典/列表几乎一一对应。`json` 模块负责在两者之间转换：

- `json.dumps(obj)`：Python 对象 → JSON 字符串（**dump s**tring）
- `json.loads(s)`：JSON 字符串 → Python 对象（**load s**tring）
- `json.dump(obj, f)` / `json.load(f)`：直接和文件打交道

```python
import json

data = {
    "name": "小明",
    "age": 18,
    "hobbies": ["篮球", "编程"],
    "is_student": True,
}

# Python 字典 → JSON 字符串
json_str = json.dumps(data, ensure_ascii=False, indent=2)
print(json_str)
# ensure_ascii=False 让中文正常显示；indent=2 美化缩进

# JSON 字符串 → Python 字典
obj = json.loads(json_str)
print(obj["name"])       # 小明
print(obj["hobbies"][0]) # 篮球
```

读写 JSON 文件：

```python
# 写入文件
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 从文件读取
with open("data.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
print(loaded)
```

> 处理中文时记得加 `ensure_ascii=False`，否则中文会变成 `\uXXXX` 的转义形式。

## 17.2 collections：更好用的容器

### Counter：计数神器

第 08 章我们用字典手动计数，`Counter` 让这件事变成一行：

```python
from collections import Counter

words = "apple banana apple cherry banana apple".split()
counter = Counter(words)
print(counter)                    # Counter({'apple': 3, 'banana': 2, 'cherry': 1})
print(counter["apple"])           # 3
print(counter.most_common(2))     # [('apple', 3), ('banana', 2)] 出现最多的前2个

# 统计字符
print(Counter("mississippi"))     # Counter({'i': 4, 's': 4, 'p': 2, 'm': 1})
```

`most_common(n)` 直接给出出现次数最多的 n 个，做词频统计极其方便。

### defaultdict：带默认值的字典

普通字典访问不存在的键会报 `KeyError`，`defaultdict` 会自动给一个默认值：

```python
from collections import defaultdict

# 按首字母把单词分组
words = ["apple", "avocado", "banana", "cherry", "cranberry"]
groups = defaultdict(list)        # 默认值是空列表
for w in words:
    groups[w[0]].append(w)        # 不用先判断键是否存在

print(dict(groups))
# {'a': ['apple', 'avocado'], 'b': ['banana'], 'c': ['cherry', 'cranberry']}
```

`defaultdict(int)` 则默认值为 0，可用来计数（等价于手动 `get(k, 0)`）。

## 17.3 os 与 pathlib：与操作系统打交道

```python
import os

print(os.getcwd())              # 当前工作目录
print(os.listdir("."))          # 列出当前目录的文件
print(os.path.exists("a.txt"))  # 文件是否存在
print(os.environ.get("HOME"))   # 读环境变量
```

不过，现代 Python 更推荐用第 12 章介绍的 `pathlib`，它更直观：

```python
from pathlib import Path

# 遍历当前目录下所有 .py 文件
for py_file in Path(".").glob("*.py"):
    print(py_file.name)

# 递归遍历所有子目录的 .txt 文件
for txt in Path(".").rglob("*.txt"):
    print(txt)
```

`glob`（当前层）和 `rglob`（递归）配合通配符 `*`，能快速找到符合模式的文件。

## 17.4 正则表达式入门

**正则表达式（Regular Expression）**是一种“描述文本模式”的强大工具，用于查找、匹配、替换符合某种规则的字符串（比如提取所有邮箱、手机号、日期）。用 `re` 模块。

### 常用元字符

| 符号 | 含义 |
| --- | --- |
| `.` | 任意一个字符（除换行） |
| `\d` | 一个数字（0-9） |
| `\w` | 一个字母/数字/下划线 |
| `\s` | 一个空白字符 |
| `+` | 前面的模式**出现 1 次或多次** |
| `*` | 前面的模式**出现 0 次或多次** |
| `?` | 前面的模式**出现 0 次或 1 次** |
| `{n}` | 恰好 n 次 |
| `[abc]` | a、b、c 中任意一个 |
| `^` / `$` | 开头 / 结尾 |
| `|` | 或 |

### re 的常用函数

```python
import re

text = "我的电话是 138-1234-5678，备用 139-8888-6666。"

# findall：找出所有匹配，返回列表
phones = re.findall(r"\d{3}-\d{4}-\d{4}", text)
print(phones)        # ['138-1234-5678', '139-8888-6666']

# search：找第一个匹配，返回匹配对象（找不到返回 None）
m = re.search(r"\d{3}", text)
if m:
    print(m.group())    # 138（第一个匹配）

# sub：替换
hidden = re.sub(r"\d{4}", "****", text)
print(hidden)        # 把 4 位连续数字替换成 ****

# split：按模式切分
print(re.split(r"[，。]", text))   # 按中文逗号句号切分
```

> 正则字符串前面通常加 `r`（原始字符串），避免反斜杠被 Python 转义，例如 `r"\d+"`。

### 一个实用例子：提取邮箱

```python
import re

text = "联系 alice@example.com 或 bob@test.org 咨询"
pattern = r"[\w.]+@[\w.]+\.\w+"
emails = re.findall(pattern, text)
print(emails)        # ['alice@example.com', 'bob@test.org']
```

模式拆解：`[\w.]+`（用户名，字母数字或点）+ `@` + `[\w.]+`（域名）+ `\.\w+`（顶级域名）。

> 正则表达式是一门“小语言”，本节只是入门。遇到复杂需求时，善用在线工具（如 regex101.com）边写边验证。

## 17.5 综合示例：分析一段文本

```python
import re
from collections import Counter

text = """Python is great. Python is easy.
I love Python and data science. Data science uses Python."""

# 提取所有单词（忽略标点），转小写
words = re.findall(r"[a-zA-Z]+", text.lower())

# 统计词频
counter = Counter(words)
print("总词数：", len(words))
print("出现最多的 3 个词：", counter.most_common(3))
```

## 本章小结

- `json`：`dumps/loads`（字符串）、`dump/load`（文件）；中文加 `ensure_ascii=False`。
- `collections.Counter` 一行完成计数，`most_common(n)` 取高频项；`defaultdict` 免去键检查。
- `os` 与 `pathlib` 操作文件系统；`glob`/`rglob` + 通配符查找文件。
- 正则表达式用 `re` 模块：`findall`（全部）、`search`（第一个）、`sub`（替换）、`split`（切分）。
- 常用元字符：`\d \w \s . + * ? {n} [] ^ $ |`；模式串加 `r` 前缀。
