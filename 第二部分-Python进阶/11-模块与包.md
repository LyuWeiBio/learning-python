## 11.1 什么是模块

一个 `.py` 文件就是一个**模块**。模块让我们能够：

- 把大程序拆成多个小文件，便于管理。
- **复用**代码：把常用函数放进一个模块，多个程序共享。
- 使用 Python **标准库**和**第三方库**里成千上万写好的模块。

Python 之所以强大，很大程度上得益于它“内置电池（batteries included）”的丰富标准库，以及庞大的第三方生态。

## 11.2 import 的几种写法

### 写法一：import 模块名

```python
import math

print(math.pi)          # 3.141592653589793
print(math.sqrt(16))    # 4.0，开平方
print(math.floor(3.7))  # 3，向下取整
print(math.ceil(3.2))   # 4，向上取整
```

导入整个模块后，用 `模块名.函数名` 的方式使用。这样最清晰，不会和自己的变量名冲突。

### 写法二：from 模块 import 具体内容

只导入需要的部分，使用时不用写模块名前缀：

```python
from math import pi, sqrt

print(pi)               # 3.141592653589793
print(sqrt(25))         # 5.0
```

### 写法三：import ... as 起别名

给模块起个简短的别名，数据科学里非常常见：

```python
import numpy as np       # 第三部分会大量这样写
import pandas as pd

# from math import sqrt as s   # 也能给函数起别名
```

> **不推荐**：`from math import *`（导入所有内容）。它会把模块里所有名字一股脑倒进来，容易和你自己的变量冲突，也让人搞不清某个名字来自哪里。

## 11.3 常用标准库速览

### math：数学函数

```python
import math

print(math.sqrt(2))        # 1.4142...
print(math.pow(2, 10))     # 1024.0
print(math.pi, math.e)     # 圆周率、自然常数
print(math.factorial(5))   # 120，阶乘
print(math.gcd(12, 18))    # 6，最大公约数
```

### random：随机数

```python
import random

print(random.random())          # [0,1) 之间的随机小数
print(random.randint(1, 6))     # 1~6 之间的随机整数（掷骰子）
print(random.choice(["石头", "剪刀", "布"]))   # 随机选一个

nums = [1, 2, 3, 4, 5]
random.shuffle(nums)            # 原地打乱
print(nums)

print(random.sample(range(1, 50), 6))   # 从 1-49 中随机取 6 个不重复的（模拟彩票）
```

### datetime：日期时间

```python
from datetime import datetime, timedelta

now = datetime.now()
print(now)                              # 当前时间
print(now.year, now.month, now.day)     # 年 月 日
print(now.strftime("%Y-%m-%d %H:%M"))   # 格式化成字符串

tomorrow = now + timedelta(days=1)      # 时间运算
print(tomorrow.strftime("%Y-%m-%d"))
```

> 标准库还有很多好东西：`os`（操作系统）、`json`（数据交换）、`re`（正则）、`collections`（高级容器）等，我们会在第 17 章专门介绍。

## 11.4 创建自己的模块

假设你写了一个 `mytools.py`：

```python
# mytools.py
PI = 3.14159

def circle_area(r):
    return PI * r ** 2

def greet(name):
    return f"你好，{name}"
```

在**同一目录**下的另一个文件里就能导入使用：

```python
# main.py
import mytools

print(mytools.circle_area(5))     # 78.53975
print(mytools.greet("小明"))       # 你好，小明

# 或者
from mytools import circle_area
print(circle_area(3))
```

## 11.5 if __name__ == "__main__"

你会在大量 Python 代码里看到这一行，它非常重要。

每个模块都有一个内置变量 `__name__`：

- 当这个文件**被直接运行**时，`__name__` 的值是 `"__main__"`。
- 当这个文件**被别的文件 import** 时，`__name__` 的值是模块名（如 `"mytools"`）。

利用这一点，我们可以把“测试代码 / 主程序入口”放在一个判断里，这样**只有直接运行时才执行，被导入时不执行**：

```python
# mytools.py
def circle_area(r):
    return 3.14159 * r ** 2

if __name__ == "__main__":
    # 这里的代码只有在直接运行 mytools.py 时才执行
    print("正在测试 circle_area：")
    print(circle_area(10))
```

- 直接运行 `python mytools.py` → 会打印测试结果。
- 在别处 `import mytools` → 不会执行测试代码，只导入函数。

> 这是 Python 项目的标准写法，请养成习惯：把程序的“启动入口”放进 `if __name__ == "__main__":`。

## 11.6 包（package）简介

当模块越来越多，可以用**包**来组织它们。包本质上就是一个**包含多个模块的文件夹**。

传统上，包目录里会有一个 `__init__.py` 文件（可以是空的），用来标识“这是一个包”：

```
myproject/
├── main.py
└── utils/                # 这是一个包
    ├── __init__.py
    ├── math_tools.py
    └── string_tools.py
```

导入包里的模块：

```python
from utils import math_tools
from utils.string_tools import clean_text
```

> 现代 Python（3.3+）即使没有 `__init__.py` 也能把文件夹当包用，但加上它更规范，也能在里面放包级别的初始化代码。初学阶段了解概念即可，第 18 章讲项目管理时会再涉及。

## 11.7 查看模块内容

想知道一个模块里有什么，可以用 `dir()` 和 `help()`：

```python
import math
print(dir(math))        # 列出 math 里所有可用的名字
help(math.sqrt)         # 查看 sqrt 的文档说明
```

善用这两个函数，你就能自己探索任何模块，而不必死记硬背。

## 本章小结

- 一个 `.py` 文件就是一个模块；`import` 让我们复用代码。
- 三种导入：`import 模块`、`from 模块 import 名字`、`import 模块 as 别名`；避免 `import *`。
- 常用标准库：`math`、`random`、`datetime`。
- 自己的模块可直接被同目录文件导入。
- `if __name__ == "__main__":` 让测试/入口代码只在直接运行时执行。
- 包是包含多个模块的文件夹，常带 `__init__.py`。
- 用 `dir()`、`help()` 探索任何模块。

## 练习题

1. **随机密码**：用 `random` 生成一个 8 位随机密码（由大小写字母和数字组成）。提示：可结合 `string` 模块的 `ascii_letters` 和 `digits`。

4. **自建模块**：创建 `calc.py`，写 `add`、`sub`、`mul`、`div` 四个函数，并在 `if __name__ == "__main__"` 里测试；再写 `use_calc.py` 导入并使用它们。
5. **理解 __name__**：在 `calc.py` 顶部加一行 `print("calc 模块被加载，__name__ =", __name__)`，分别“直接运行 calc.py”和“运行 use_calc.py”，观察输出有何不同并解释原因。

**第 1 题**：

```python
import random
import string

chars = string.ascii_letters + string.digits   # 大小写字母 + 数字
password = "".join(random.choice(chars) for _ in range(8))
print(password)
```

**第 2 题**：

```python
import random

count = 0
times = 1000
for _ in range(times):
    total = random.randint(1, 6) + random.randint(1, 6)
    if total == 7:
        count += 1
print(f"点数和为 7 的比例：{count / times:.2%}")   # 理论值约 16.7%
```

**第 3 题**：

```python
from datetime import date

today = date.today()
target = date(2030, 1, 1)
print(f"还有 {(target - today).days} 天")
```

**第 4 题**：

```python
# calc.py
def add(a, b): return a + b
def sub(a, b): return a - b
def mul(a, b): return a * b
def div(a, b): return a / b if b != 0 else "除数不能为0"

if __name__ == "__main__":
    print(add(2, 3), sub(5, 2), mul(3, 4), div(10, 2))
```

```python
# use_calc.py
import calc
print(calc.add(10, 20))     # 30
print(calc.div(9, 3))       # 3.0
```

**第 5 题**：
- 直接运行 `calc.py` 时输出 `__name__ = __main__`，并会执行 `if` 块里的测试。
- 运行 `use_calc.py` 时，import 会加载 calc，输出 `__name__ = calc`，但**不会**执行 calc 的 `if __name__ == "__main__"` 块。
- 原因：被导入时 `__name__` 是模块名而非 `"__main__"`。
